# Subscription Management Platform

Live Frontend: https://sec913-academic-project-gt0zl084n-aswith27s-projects.vercel.app

## Project Architecture

- React + Vite (Frontend)
- FastAPI (API Gateway)
- Spring Boot (Core Backend)
- PostgreSQL (Relational Database)
- Node.js + Express (Activity Service)
- MongoDB (Activity Logs)
